# Knowledge index

Оглавление проектных знаний для агентов. `worker` читает этот файл перед реализацией и подгружает релевантные темы по контексту задачи.

## Чем отличается от TECH.md

- `TECH.md` — компактная карта проекта (стек, команды, базовые правила). Читается каждый раз.
- `knowledge/*.md` — детальные подсказки по узким темам (паттерны Yii2, стиль кода, нюансы миграций и т.п.). Читается выборочно.

Когда заметка вырастает из абзаца в страницу — это сигнал вынести её в отдельный `knowledge/<тема>.md` и добавить сюда строку.
