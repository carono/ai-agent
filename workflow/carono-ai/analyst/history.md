# Analyst — журнал прогонов

Хронологический лог прогонов скилла `analyst` в проекте `carono-ai`. Каждый блок — одна сессия. Скилл читает этот файл при следующих запусках, чтобы:
- не предлагать повторно отклонённое (молча)
- упоминать прошлые отклонения только если пользователь сам поднимает связанную тему
- проверять выполнение условий возврата для отложенных идей
